# Estruturas de Repetição

3 pilares das estruturas de repetição.
```
1. inicialização
2. verificação
3. atualização
```

```java
// inicialização
int estoque = sc.nextInt(); 

// verificação
while( estoque != 0 ) {
  System.out.println("Estou vendendo");
  
  estoque--; // atualização
  // estoque -= 1
  // estoque = estoque - 1
}
```
